# </> How to run

```bash
# Clone Repository
$ git clone https://github.com/dfilastro/Rocketseat-Ignite.git
```

```bash
$ cd /"Chapter III"
$ cd /"II - Challenge"
# Install Dependencies
$ yarn
```

```bash
$ yarn dev
```

Go to http://localhost:3000/ to see the result.
